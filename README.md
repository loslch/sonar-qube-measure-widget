# Sonar Qube Measure Widget

## Quick start

Change Sonar Qube configurations on `get_measure.sh` (e.g. `SONAR_API_URL`, `SONAR_PROJECT_KEY`, `SONAR_TOKEN`).

```sh
$ npm install
$ npm run out
```

Then, store the output as html.
